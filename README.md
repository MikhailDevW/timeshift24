### Installation guide
First you need fork and clone  
Install venv:
```
python -3.9 -m venv venv
```
Activate them
```
source venv/Scripts/activate
```
Install requirements
```
pip install -r requirements.txt
```
Aaand run...
```
python manage.py runserver
```
